# Place Your Images Here

## How to Use Images

1. **Profile Picture**: 
   - Place your profile image here with filename: `profile.jpg` or `profile.png`
   - Recommended size: 400x400px (square)

2. **About Section Image**:
   - Place your about image here with filename: `about.jpg` or `about.png`
   - Recommended size: 450x350px

3. **Supported Formats**: jpg, jpeg, png, webp, gif

4. **Update Component References**:
   - Hero component will use: `/images/profile.jpg`
   - About component will use: `/images/about.jpg`

Example file structure:
```
public/
├── images/
│   ├── profile.jpg    (Your profile picture)
│   └── about.jpg      (Your about section picture)
```

Then restart your development server with `npm run dev`
